## Available MCP Servers
Common MCPs you can add via .mcp.json:

- **Google Calendar** — Read/write calendar events
- **Gmail / IMAP** — Read and send emails
- **Google Drive** — Read/write files in Drive
- **Notion** — Read/write Notion pages and databases
- **Linear** — Project management tasks and issues
- **Slack** — Read/send Slack messages
- **Reminders** — macOS Reminders integration

See .mcp.json.example for configuration format.
